# Sales Dashboard Analysis

## Overview

This project focuses on analyzing sales data and building a simple dashboard to understand business performance. The main goal was to track sales trends, identify top-performing products, and get a clear view of overall business health.

Instead of just working with raw data, I tried to turn it into something that can actually help in decision-making.

## What this project is about

* Tracking overall sales performance
* Identifying top and low-performing products
* Understanding customer buying patterns
* Visualizing key business metrics through a dashboard

## Tools used

* Excel / Power BI
* Python (for data cleaning, if needed)
* Pandas
* Matplotlib / Seaborn

## Workflow

1. Data collection and inspection
2. Data cleaning and formatting
3. Sales analysis by category and product
4. KPI calculation
5. Dashboard creation
6. Insights extraction

## Key insights

* A small number of products generate most of the revenue
* Some categories consistently outperform others
* Sales fluctuate based on time period and product type
* Certain products have low sales despite high availability

## Files in this repo

* sales_analysis.ipynb / Excel file
* dashboard.pbix or Excel dashboard
* dataset.csv
* CASE_STUDY.md

## What I learned

This project helped me understand how raw sales data is used in real business reporting. I also improved my skills in creating dashboards that are simple but useful for decision-making.
